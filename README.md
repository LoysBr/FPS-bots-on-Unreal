# FPS Game - Unreal Engine Experimental Project

A small-scale FPS game built in UE5 to explore modern game development practices, Agentic AI workflows, and maintain a collection of code and blueprint examples I can easily refer back to or reuse.


## 🎯 Main Project Goals

- Learn UE5 main features from my angle of Unity Programmer
- Especially learn about network and AI Behavior systems
- Explore AI agents development workflows in UE 
- Deploy a Windows Offline demo
- Deploy a Windows Multiplayer Demo 


## 💻 Technical Focus

Learning everything in Unreal probably takes years, so I am focusing first on some major elements :
- Deeply understand the Network system and how to create clean and optimized Multiplayer architectures
- Deeply understand the AI Controller / Behavior Tree system 
- Learn how to use C++ to provide Gameplay, in combination with Blueprints 
- Learn about the Gameplay Ability System
- Get basic knowledge of everything


## 🎮 Game Design

For now I chose to have a very basic deathmatch FPS
Simple AI Bots, but giving a correct game feel (neither too easy nor too good)
Multiplayer support, both in LAN and Online.

## 🚀 Get Started - Download

- Clone the repository and open the project in **Unreal Engine 5.7.4** or newer.
- The **Unreal project name is currently "ZulaOnUE"** (a reference to Zula Strike - I was a core member of its development team for over 5 years), maybe I'll rename it later.
- Open `Content/Levels/Lvl_ZulaOnUE_Level1.umap`. A .exe build will come soon.
